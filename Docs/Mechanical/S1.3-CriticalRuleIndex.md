## Classification Rules

### Mechanical Criticality Types

- **Safety** — a rule whose failure could create unsafe geometry, unsafe motion, or unacceptable release risk
- **Function** — a rule whose failure would prevent the product from performing its intended build or adjustment behavior
- **Manufacturing** — a rule whose failure would break fabrication readiness, export readiness, or manufacturable definition

### Validator Enforcement Tags

- **Strict** — rule should be enforced by deterministic validator logic with explicit pass/fail behavior
- **Standard** — rule should be validated, but strict blocking enforcement is not yet required by the current baseline

### Risk Priority Scale

- **High** — failure would materially block build validity, adjustment validity, or manufacturing handoff
- **Medium** — failure would materially degrade traceability or downstream quality, but may not immediately block all flows
